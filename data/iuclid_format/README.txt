IUCLID 6 format bundle (committed copy)
========================================

Place the extracted ECHA "IUCLID 6 9.0.0_format" (or compatible) tree here as:

  IUCLID_6_9_0_0_format/

This directory supplies phrase / picklist / XSD resources for decoding IUCLID codes in
offline dossiers. It is the official-style format package from ECHA downloads — not the
REACH dossier bulk.

REACH substance coverage still depends entirely on what is inside
data/reach_demo/reach_subset.zip (or your OFFLINE_LOCAL_ARCHIVE). That dossier zip is a
demo subset unless you replace it with a full local archive (not committed to GitHub).

See data/echa_cloud/README.txt and the main README section "IUCLID offline data for Streamlit Cloud".
